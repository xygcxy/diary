An Introduction To PostCSS

Frontend

一个相对较新的工具postcss。postcss旨在重塑CSS自定义插件和工具的生态系统。随着预处理如SASS，它将扩展语法和功能到现代浏览器友好的CSS。 怎么处理的？JavaScript。 JavaScript有改变我们的风格比其他处理器更快的能力。使用javascript去操作css。

通常通过grunt和gulp去下载postcss的插件进行使用。早期的类似Autoprefixer。

gulp使用：

var postcss = require('gulp-postcss');

gulp.task('styles', function () {
    return gulp.src('path/to/dev/style.css')
        .pipe(postcss([]))
        .pipe(gulp.dest(path/to/prod))
});

grunt配置：

module.exports = function(grunt) {
  grunt.initConfig({
    pkg: grunt.file.readJSON('package.json'),
    styles: {
      options: {
        processors: []
      },
      dist: {
        files: [{
          expand: true,
          cwd: 'dev/',
          src: ['**/*.css'],
          dest: 'prod/'
        }]
      }
    }
  });

  // Load post-css.
  grunt.loadNpmTasks('grunt-postcss');

};
