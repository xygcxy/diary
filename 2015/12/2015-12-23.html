An in-depth Look at CORS

Frontend

CORS是一种相对较新的HTML5 API,它允许我们的网站请求外部资源。它不同与传统同源限制策略,让我们能够请求资源在不同的域名下。

Internet Explorer 8和9支持CORS只有通过XDomainRequest类。IE10和Android的默认浏览器4.3时在使用<canvas>元素时使用new XMLHTTPRequest不能跨域。

简单跨域例子： 

var xhr = new XMLHttpRequest();
    var url = "//example.com";
    xhr.open("GET", url);
    xhr.onreadystatechange = function() {

        if (xhr.status === 200 &#038;&#038; xhr.readyState === 4) {

            document.querySelector("body").innerHTML = xhr.responseText
        }
    }
    xhr.send();

需要在服务器端添加header信息如： 

< ?php
header("Access-Control-Allow-Origin: *")
?>

或者在apache配置里设置header。 注意，在HTML里添加以下代码是无效的： 

<meta http-equiv="Access-Control-Allow-Origin" content="*">

因为信息头是服务器才能设定，纯HTML是无法作用的。
