ES7 Proposal: The Pipeline Operator

Frontend

这个提议引入了一个新的操作符|>, 同F#, OCaml, Elixir, Elm, Julia, and LiveScript, ,以及UNIX管道相似。它是向后兼容的方式简化函数调用链以一种可读的,功能的方式,并提供一个实用的选择扩展内置的原型。例如： sqrt(64)与64 |> sqrt相等. 给定下面的代码： 

function doubleSay (str) {
  return str + ", " + str;
}
function capitalize (str) {
  return str[0].toUpperCase() + str.substring(1);
}
function exclaim (str) {
  return str + '!';
}

下面的调用是等价的: 

let result = exclaim(capitalize(doubleSay("hello")));
result //=> "Hello, hello!"

let result = "hello"
  |> doubleSay
  |> capitalize
  |> exclaim;

result //=> "Hello, hello!"
