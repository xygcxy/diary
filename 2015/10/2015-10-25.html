纯CSS3实现的叉叉按钮

标签（空格分隔）： Frontend

CSS3实现的叉叉按钮，利用了CSS3里的transform属性，rotate交叉倾斜45度角实现。

只试用span标签即可实现，HTML代码：

<span class="close hairline"></span>

CSS代码如下：

.close {
    position: relative;
    display: inline-block;
    width: 50px;
    height: 50px;
    overflow: hidden;
}
.close.hairline::before, .close.hairline::after {
    height: 5px;
}
.close::before {
    -webkit-transform: rotate(45deg);
    -moz-transform: rotate(45deg);
    -ms-transform: rotate(45deg);
    -o-transform: rotate(45deg);
    transform: rotate(45deg);
}
.close::before, .close::after{
    content: '';
    position: absolute;
    height: 2px;
    width: 100%;
    top: 50%;
    left: 0;
    margin-top: -1px;
    background: #000;
}
.close::after {
    -webkit-transform: rotate(-45deg);
    -moz-transform: rotate(-45deg);
    -ms-transform: rotate(-45deg);
    -o-transform: rotate(-45deg);
    transform: rotate(-45deg);
}

其中第二个CSS中的height代表线的粗细程度。span标签需设置 position: relative 和 display: inline-block来显示行内元素。 同时利用after和before来显示两条交叉线，伪类定位需设置position:absolute。 交叉线就是rotate(45deg)和rotate:(-45deg)。

变花样：class加一个值设定其::before:width:200%;left:-50%,可以得到一个四角是尖的叉叉。也可以直接设置border-radius获得一个圆润的叉叉。
