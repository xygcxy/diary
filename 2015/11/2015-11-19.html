The Difference Between Responsive and Adaptive Design

Frontend

自适应和适应设计： 响应网站和自适应站点表现是一样的，他们都在浏览器不同窗口下表现的一种设计（常见准则：浏览器的宽度）。

自适应网站不管浏览器的宽度是什么，根据浏览器窗口上调整布局（和功能）。布局会作出相应的反应。 eg： HTML:

<div class="module">
  <h1>This is a reponsive container</h1>
  <p>This container is <span class="element"></span> pixels wide when the browser is <span class="browser"></span> pixels wide. Resize the browser to see it change.</p>
</div>
html {
  background: #333;
}

CSS:

body {
  display: flex;
  justify-content: center;
  align-items: center;
}

.module {
  background-color: white;
  border-radius: 10px;
  padding: 20px;
  width: 80%;
}

JS:

$( ".element" ).html( $( ".module" ).width() );
$( ".browser" ).html( $( window ).width() );

$( window ).resize(function() {
  $( ".element" ).html( $( ".module" ).width() );
  $( ".browser" ).html( $( window ).width() );
});

适应性网站适应浏览器的宽度在一个特定的点。换句话说，该网站只关注浏览器在某个特定的宽度，在这个点上是适应布局。 适应性网站主要实现是通过媒体查询来实现。 例如：网易有道在线web端翻译就是在小屏幕（1024*768）下，广告位离底部距离缩小！
