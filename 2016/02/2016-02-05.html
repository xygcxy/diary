outHTML选择器

Frontend

据MDN这个属性“outerHTML属性可以获取该DOM元素及其后代节点所形成的序列化的HTML片段，改变outerHTML属性可以用来把所属元素替换成为给定字符串解析后的DOM节点。下面的HTML: 

<div id="parent">
  <p>Some text for parent.</p>
  <div id="child">
    <div id="grandChild">
      <p>Some text for grandchild.</p>
    </div>
    <p>Some text for child.</p>
  </div><!-- #child -->
</div><!-- #parent -->

你可以这样使用来操作子元素： 

console.log(child.outerHTML);

/* will log:
<div id="child">
  <div id="grandChild">
    <p>Some text for grandchild.</p>
  </div>
  <p>Some text for child.</p>
</div>
*/

grandChild.outerHTML = '<p>New paragraph</p>';
console.log(document.getElementById('grandChild')); // null

您可以在任何元素上使用这个属性，你可以得到根元素的innerHTML，但你不能改变它;这将抛出一个错误。
