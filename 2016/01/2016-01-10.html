Frontend

7.You can use invalid characters as delimiters in your HTML class lists
为了是你的代码容易扫描和阅读。可以再class属性中加入字符：让其无效： 

<div class="col col-4 col-8 ** c-list ** bx bx--rounded bx--transparent">

<div class="col col-4 col-8 || c-list || bx bx--rounded bx--transparent">

<div class="col col-4 col-8 && c-list && bx bx--rounded bx--transparent">

如果非要有效可以加转义符\.还有的class可以使用Unicode特征的来作为属性名。如 

<div class="♥ ★"></div>

8.Animation iterations can be fractional values
animation-iteration-count属性是整数值的例子会告诉动画运行多少次。但也许你不知道,您可以使用分数值: 

.example {
  animation-iteration-count: .5;
}

让动画运行一半。
但是这个视觉效果跟时间函数有关，例如你使用ease视觉上会超过一半。

9.Animation shorthand can break because of the animation’s name
定义动画名称需要注意。不能定义有动画内部定义的属性值。

10.You can select ranges of elements
看以下代码： 

ol li:nth-child(n+7):nth-child(-n+14) {
  background: lightpink;
}

这段代码表示选中第7到14行的标签。有时会很有用。详情请看这里.

12.Some attribute values are case insensitive in selectors 

div[class="BOX"] {
  color: blue;
}

input[type="EMAIL"] {
  border: solid 1px red;
}

请注意属性值都是大写的情况。在这种情况下。class将不起作用,因为类属性是区分大小写的。另一方面,type值是不区分大小写的。
