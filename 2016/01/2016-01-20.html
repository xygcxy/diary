Function Definitions in JavaScript

Frontend

函数声明：
声明一个函数声明。 

function add(a, b) {
    return a + b;
}

var x = add(1, 2);
// x = 3

function add(a, b) {
    return a + b;
}

函数表达式： 

var add = function(a, b) {
    return a + b;
};

也可以给其命名： 

var add = function add(a, b) {
    return a + b;
};

函数表达式不能提升变量，所以： 

var x = add(1, 2);
// TypeError: add is not a function

var add = function add(a, b) {
    return a + b;
};

Arrow Functions：

var add = (a, b) => a + b;

类似函数表达式，箭头功能不提升函数——只有函数声明。： 

const numbers = [1, 2, 3];
const doubled = numbers.map(n => 2 * n);

箭头功能不能有名字。箭头功能语法非常简洁,适合简单、简短的函数。它不允许函数名。
箭头功能词法绑定当前这个值。
