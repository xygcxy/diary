Counters in CSS： 

<ol class="list">  
    <li>a</li>
    <li>b</li>
    <li>c</li>
</ol>  
.list {
    counter-reset: i; //reset conunter
}
.list > li {
    counter-increment: i; //counter ID
}
.list li:after {
    content: "[" counter(i) "]"; //print the result
}
