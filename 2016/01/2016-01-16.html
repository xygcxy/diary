JavaScript对象遍历属性和值

Frontend

加入你输出来一个对象，但是苦于不知道里面有哪些属性和值，这个时候，你可以通过下面的代码来遍历这个对象，从而输出全部的属性和值。

一个是通过for循环来输出，一个则是通过for……in。

var len = obj.length;
for(var i=0; i<len; i++){
    console.log(obj[i]);
}

第二个则是for in

for(var p in obj){
// 方法
    if(typeof(obj[p])=="function"){
        obj[p]();
    }else{
        // p 为属性名称，obj[p]为对应属性的值
        props+= p + "=" + obj[p]+"<br />";
    }
}
// 最后显示所有的属性
console.log(props);
