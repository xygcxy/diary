csswizardry-grids – By Harry Roberts, so it must be good. xr – Like I said, I love simple things that do one thing well, so this “wrapper around XMLHttpRequest” makes the list. Bootply – Bootstrap is always a popular subject so this Bootstrap editor and playground is worth checking out. ES Feature Tests – “Feature test ES6+ JavaScript syntax and APIs. Use these in-browser test results to deliver the best code to each browser. Load your native ES6+ code in ES6+ compliant browsers and your transpiled code in older browsers.” You Might Not Need jQuery Plugins – A directory of jQuery-free plugins and scripts that can be filtered according to IE support. tota11y – “Helps visualize how your site performs with assistive technologies.” Where is it? – A Chrome extension that “makes code navigation on GitHub easier. Look up and jump around class/method definitions with a single click.” rucksack – “Makes CSS development fun again, with features the language should have come with out of the box. It’s built on PostCSS, it’s modular, it doesn’t add any bloat, and it’s lightning fast.” Microm – Lets you convert an audio stream to the user’s microphone into a downloadable audio file (MP3, WAV, base64). SVG Path Builder – You’ll have to play with this one to get used to it, but it looks like a really nice way to create custom SVG graphics
